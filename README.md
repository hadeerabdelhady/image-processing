# Digital Image Processing

Digital Image Processing is a field of study that focuses on the manipulation and analysis of images using computational techniques. It aims to improve image quality, extract useful information, and prepare images for further analysis in computer vision and related applications.

In this course, several fundamental topics were covered. **Image Enhancement** techniques were studied in both the **spatial domain** and the **frequency domain**, focusing on improving visual quality and highlighting important features in images. The course also included **Image Restoration**, which deals with recovering degraded or noisy images using mathematical and filtering approaches.

Another key topic was **Image Segmentation**, which involves dividing an image into meaningful regions or objects to simplify analysis. Finally, **Mathematical Morphology** was explored, providing tools for processing shapes and structures in binary and grayscale images using operations such as dilation and erosion.

This repository includes assignments and implementations that demonstrate these core concepts in digital image processing.
